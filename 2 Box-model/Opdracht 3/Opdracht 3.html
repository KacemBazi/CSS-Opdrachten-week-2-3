Opdracht 3

Wat gebeurt er als je een width en een height toevoegt aan de span tag ( <span> ) van vorige opdracht?

Dit is een Block Box model die de hele regel in beslag neemt waardoor de volgende Block Box op de volgende regel terechtkomt

In dit geval is het bold gedeelte een inline tekst Dit is een inline voorbeeld nu kun je zie dat de tekst niet verschoven is naar de volgende regel

Het antwoord is dat de width en heitght geen invloed heeft op een inline.
