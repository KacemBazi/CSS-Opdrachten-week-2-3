Opdracht 2

CSS pas je toe met een bepaalde syntax.
Hoe ziet zo'n syntax eruit?
Kun je ook uitleggen wat elk element uit de syntax betekent?



- Een syntax is een werkvolgorde/structuur.

- Je gebruikt een "selector" om aan te geven welk element je wilt veranderen, in dit geval gaat het om h1 tag.

- Vervolgens open je de "declaration" met een accolade { teken.

- De "decleration" vul je dan met de properties die je wilt veranderen en de waardes waarmee je deze wilt veranderen.

- De "decleration" scheidt je met een punt komma (;) teken

- Vervolgens sluit je met een accolade }.
