● type selectors
● descendant selectors

Type selectors

Een type selector komt overeen met de naam van een type documenttaalelement.
Een type selector komt overeen met elk exemplaar van het elementtype in de documentboom.

Descendant selectors

Soms wil je dat selectors overeenkomen met een element dat de afstammeling is van een ander element in de documentboom
(bijv. " De <em> elementen die deel uitmaken van een <h1> element").
Descendant selectors drukken zo'n relatie uit in een patroon.
Een afstammende selector bestaat uit twee of meer selectors gescheiden door witruimte .
Een afstammeling selector in de vorm " A B " matchen wanneer element B een willekeurige afstammeling van enkele voorouder element A

Class selectors

De CSS class selector komt overeen met elementen op basis van de inhoud van hun classattribuut.
Om dit te doen begint je met de naam van het element en schrijft je vervolgens het punt (.) - teken,
gevolgd door de naam van de class
